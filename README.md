# SudokuMNIST Multi-Task CNN

A compact multi-task convolutional neural network implemented in PyTorch for solving three different prediction tasks on the SudokuMNIST dataset.

## Overview

This project explores how convolutional neural networks can learn structured relationships between visual objects while operating under a strict parameter budget. The model is trained on SudokuMNIST, a synthetic dataset generated by arranging MNIST digits in a 3×3 grid.

The network learns a shared visual representation and simultaneously solves three tasks:

1. **Missing Digit Prediction**
   - Predict the digit (0–9) that does not appear in the grid.
   - 10-class classification problem.

2. **Row/Column Order Classification**
   - Determine whether each row and column is:
     - Unordered
     - Strictly ascending
     - Strictly descending
   - Six independent 3-class classification outputs.

3. **Row/Column Sum Prediction**
   - Predict the normalized sums of all rows and columns.
   - Six-output regression problem.

---

## Dataset

### SudokuMNIST

Each sample is generated by:

- Selecting 9 distinct digits from 0–9.
- Arranging them randomly in a 3×3 grid.
- Rendering each digit using an MNIST image.
- Producing an 84×84 grayscale image.

The remaining digit becomes the target for the missing-digit task.

### Input

- Shape: `1 × 84 × 84`
- Grayscale image

### Outputs

| Task | Type | Output Size |
|--------|--------|--------|
| Missing Digit | Classification | 10 |
| Sort Order | Classification | 6 × 3 |
| Row/Column Sums | Regression | 6 |

## Model Architecture

The model consists of:

- Shared convolutional feature extractor
- Parameter-efficient CNN backbone
- Task-specific prediction heads
    - Missing digit head
    - Sorting classification head
    - Sum regression head

The architecture is designed to satisfy the project constraint of fewer than **20,000 trainable parameters**.
